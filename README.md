Ghari
=====

Ghari will be another attempt to create a website builder in nodejs

### Hot to run locally?

1. Clone repository
```sh
$ npm install
$ bower install
```
1. Install
```sh
$ npm install
$ bower install
```
1. Run server
```sh
$ gulp server
```
1. Open URL: http://localhost:8000
```sh
$ open http://localhost:8000
```
